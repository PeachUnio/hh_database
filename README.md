# Приложение для создания базы с вакансиями с Hh.ru
### Приложение создает базу данных и предоставляет пользователю необходимую информацию.
### Статус: ~в разработке~
### Версия: 1.0
# Использование
### Для работы программы необходимо установить следующие зависимости:
* ### `requests (>=2.32.5,<3.0.0)`
* ### `python-dotenv (>=1.2.1,<2.0.0)`
* ### `psycopg2 (>=2.9.11,<3.0.0)`
### Язык программирования: `>=3.13`
### Не забудьте указать нужные данные в `.env.exsample`
### ! Не забудьте удалить `.exsample` для корректной работы программы !
